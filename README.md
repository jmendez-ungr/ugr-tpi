# TPI Ventas - Streamlit Demo

App de Streamlit para mostrar una herramienta complementaria del TPI modelo de ventas.

## Archivos
- `streamlit_app.py`: aplicación principal
- `requirements.txt`: dependencias

## Ejecución local
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Fuente de datos
La app consulta la base directamente desde Google Sheets.
